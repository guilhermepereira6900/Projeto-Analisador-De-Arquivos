# Projeto-Analisador-De-Arquivos

Projeto onde desenvolvi um script usando python onde apenas de dar o diretório do seu arquivo, o script reconhece o tipo do arquivo e o armazena no seu devido diretório.

# Sobre o Projeto

Este projeto foi desenvolvido como parte do módulo de gerenciamento de arquivos do curso python automático. O objetivo principal era aprender a manipular arquivos usando python.

# Tecnologias Utilizadas
* **Linguagem:** Python 3.10
* **Framework/Lib:** pathlib, shutil

# Como Executar o Projeto

Para rodar este projeto localmente, siga os passos abaixo:
 baixe os arquivos .py e use python como interpretador, os restantes dos arquivos são apenas exemplos usados no projeto.
